# MW Cast Frontend

A modern, high-performance frontend for the MW Cast live streaming platform, built with React, Tailwind CSS, and Vite.

## Tech Stack

- **React 18** - UI Library
- **Tailwind CSS** - Styling
- **Vite** - Build Tool
- **React Router** - Routing
- **Lucide React** - Icons
- **Framer Motion** - Animations

## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Navigate to the frontend directory:
   ```bash
   cd frontend
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

## Project Structure

- `src/components` - Reusable UI components
- `src/pages` - Page components (Home, Stream, etc.)
- `src/layouts` - Layout wrappers
- `src/assets` - Static assets
- `src/index.css` - Global styles and Tailwind directives
